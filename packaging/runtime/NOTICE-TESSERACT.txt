Tesseract OCR runtime
=====================

JustSearch may bundle Tesseract OCR for baseline scanned-document text extraction.

Runtime:
- Tesseract OCR 5.5.0.20241111
- License: Apache License 2.0
- Source: https://github.com/tesseract-ocr/tesseract/releases/download/5.5.0/tesseract-ocr-w64-setup-5.5.0.20241111.exe

Language data:
- eng.traineddata from tesseract-ocr/tessdata_fast
- License: Apache License 2.0
- Source: https://github.com/tesseract-ocr/tessdata_fast

The pinned source URL, SHA-256 values, file sizes, and validation command are declared in
packaging/runtime/tesseract-windows.v1.json.
