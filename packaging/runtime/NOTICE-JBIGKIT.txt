JBIG-KIT (libjbig) — GNU GPL v2 Notice & Written Offer
======================================================

JustSearch's Windows OCR bundle includes "libjbig-0.dll", part of JBIG-KIT,
a lossless image-compression library by Markus Kuhn. It is reached only as an
optional JBIG codec inside libtiff (used by Leptonica/Tesseract for OCR), is
dynamically linked, and is separately replaceable.

Replaceability
--------------
"libjbig-0.dll" is a standalone dynamically-linked library. It may be replaced
or removed by the user; the rest of JustSearch and the OCR pipeline continue to
function (JBIG is a rare bilevel-image codec; non-JBIG TIFFs and all other OCR
input are unaffected).

This file is authored alongside the tempdoc-632 native-binary disposition gate.
The JBIG-KIT artifact is recorded as `strong-copyleft` in
packaging/runtime/tesseract-bundled-libraries.v1.json with a dated acceptance in
config/native-license-acceptances.json. If JustSearch later ships a Tesseract
build without JBIG, this file, that manifest row, and that acceptance entry are
removed together.
