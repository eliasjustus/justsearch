# Keyboard Shortcuts

## Global

| Shortcut | Action |
|----------|--------|
| Ctrl+K | Open search (focus search bar) |
| Ctrl+, | Open settings |
| Escape | Close current panel / clear search |

## Search results

| Shortcut | Action |
|----------|--------|
| Enter | Open selected result |
| Up/Down | Navigate results |
| Tab | Switch between search and results |

## General navigation

| Shortcut | Action |
|----------|--------|
| Ctrl+1 | Switch to Search view |
| Ctrl+2 | Switch to Library view |
| Ctrl+3 | Switch to Settings view |
